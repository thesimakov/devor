# Локализация (RU / TJ)

Строки разнесены по **модулям** (один файл — одна область интерфейса):

| Файл        | Содержимое                          |
|------------|--------------------------------------|
| `common`   | города, валюта, «договорная», км     |
| `header`   | шапка, поиск, тосты чата             |
| `footer`   | подвал сайта                       |
| `home`     | главная страница                   |
| `map`      | блок карты объявлений              |
| `how`      | блок «как пользоваться»            |
| `account`  | мои объявления, избранное          |

Для каждого языка — папка `ru/` или `tj/`. Сборка словаря: `lib/i18n.js` (`deepMerge`).

Выбор языка: `LanguageProvider` в `_app.js`, ключ `localStorage`: `devor_lang` (`ru` | `tj`).

В компонентах: `useLanguage()` → `t('раздел.ключ', { var: '…' })`.
